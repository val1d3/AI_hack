Установка

0) Клонируйте репозиторий: `git clone https://github.com/Apdite/AI_hack.git`
1) Установите зависимости: `pip install -r requirements.txt`

Структура проекта

* models/ : Скрипты для моделей (YOLO, MMDetection).
* ui/ : Интерфейсы на Gradio и Streamlit.
* tests/ : Тесты.
* data/ : Датасеты.
* configs/ : Конфигурации.
